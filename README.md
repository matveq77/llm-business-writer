# AI Memo Agent

Интеллектуальный агент, который превращает текстовые запросы в официальные приказы и структурированные данные.

## Суть проекта
Пользователь пишет фразу: *"Подготовь приказ о командировке в Варшаву на 3 дня для инженера Петрова"*. 
Агент автоматически генерирует:
1. **Файл .docx** — готовый к печати документ.
2. **JSON-объект** — структурированные данные для интеграции в ERP/CRM системы.

## Стек технологий
- **Backend:** Python (FastAPI) — логика и API.
- **LLM:** Google Gemini 1.5 Flash (или DeepSeek) — извлечение сущностей через Prompt Engineering.
- **Docs:** `python-docx` — генерация документов.
- **Frontend:** Streamlit — интерфейс чат-бота.

## Как работает
1. **Анализ:** LLM парсит текст и возвращает строгий JSON (имя, город, даты).
2. **Генерация:** Бэкенд подставляет данные в шаблон и создает `.docx`.
3. **Результат:** Пользователь скачивает файл и видит извлеченные реквизиты в UI.
